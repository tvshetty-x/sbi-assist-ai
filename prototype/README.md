# SBI AssistAI Prototype

This folder contains the UI prototype screens for **SBI AssistAI**, an AI-powered multilingual banking assistant designed to make digital banking easier, safer, and more accessible for everyone.

The prototype demonstrates how users can interact with the assistant through text or voice, receive banking guidance in their preferred language, and learn about safe banking practices.

## Prototype Flow

### 01 - Welcome Screen

**File:** `01-welcome-screen.png`

The entry point of the application. It introduces SBI AssistAI and highlights the core features such as voice banking, AI assistance, multilingual support, and secure banking.

---

### 02 - Language Selection Screen

**File:** `02-language-selection.png`

Allows users to choose their preferred language before interacting with the assistant. This helps improve accessibility for users who are more comfortable in regional languages.

Supported languages include:

* English
* Hindi
* Kannada
* Tamil
* Telugu

---

### 03 - AI Chat Screen

**File:** `03-ai-chat-screen.png`

Demonstrates a real banking conversation where a customer asks how to reset their YONO password. The assistant provides clear step-by-step guidance along with security reminders.

---

### 04 - Voice Assistant Screen

**File:** `04-voice-assistant-screen.png`

Shows how users can interact with SBI AssistAI using voice commands. This feature is especially useful for senior citizens, first-time smartphone users, and customers who prefer speaking over typing.

---

### 05 - Fraud Awareness Screen

**File:** `05-fraud-awareness-screen.png`

Provides important fraud prevention tips and banking safety guidance. The screen educates users about common scams such as OTP fraud, phishing attempts, and fake banking calls.

---

## Goal of the Prototype

The goal of this prototype is to demonstrate how AI can help increase digital banking adoption by:

* Simplifying banking interactions
* Supporting multiple languages
* Providing voice-based assistance
* Improving fraud awareness
* Making banking more accessible for all users

This prototype was developed as part of the **SBI Hackathon @ GFF 2026**.
